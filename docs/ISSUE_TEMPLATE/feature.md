Cada tarea debe tener:

- Título claro

- Descripción corta

- Responsable

- Relación con una épica de Jira

## Ejemplo de Issue:

Título: Crear formulario de registro de voluntarios

Descripción:
Diseñar e implementar el formulario de registro para voluntarios,
incluyendo validaciones básicas.

Criterios de aceptación:
- Campos obligatorios
- Validación de correo
- Mensaje de confirmación
